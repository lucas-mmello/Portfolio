Desenvolvimento Web

Tenho experiência no desenvolvimento de sites e aplicações web utilizando JavaScript e PHP com Laravel no back-end. Além disso, já trabalhei com ASP.NET em C#, expandindo meus conhecimentos em diferentes tecnologias para construção de sistemas robustos.

No mundo JavaScript, utilizo TypeScript para tornar o código mais seguro e escalável. No front-end, além de HTML, CSS e JavaScript, faço uso de ferramentas que otimizam o desenvolvimento, como SASS, Bootstrap e React.

No back-end, além do Laravel, já desenvolvi APIs utilizando Node.js com Express. Para bancos de dados, trabalho com MySQL, PostgreSQL e MongoDB, utilizando Mongoose para modelagem no Node.js.

Estou sempre buscando aprimorar minhas habilidades e explorar novas tecnologias para tornar meus projetos mais eficientes e performáticos.

Desenvolvimento Mobile

Tenho experiência no desenvolvimento de aplicativos móveis utilizando React Native, aliado ao TypeScript para garantir um código mais estruturado e seguro.

Utilizo o Expo para facilitar o desenvolvimento e a prototipação rápida de aplicativos, além de bibliotecas essenciais para otimizar a performance e a usabilidade das aplicações.

Para a estilização, utilizo Styled Components, garantindo flexibilidade e organização no código. Além disso, trabalho com diversas bibliotecas do ecossistema do React Native, como React Navigation para navegação, Axios para requisições HTTP e AsyncStorage para gerenciamento de estado, entre outras.

Estou sempre explorando novas tecnologias e práticas para aprimorar a experiência do usuário e desenvolver aplicativos eficientes e escaláveis.
